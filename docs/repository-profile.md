# GitHub 仓库资料建议

创建 GitHub 仓库后，可以按下面内容配置仓库资料。

## 仓库名称

```text
open-source-component-catalog
```

如果希望使用中文名称，也可以使用：

```text
component-catalog-cn
```

## 仓库描述

```text
按模块整理可拼装项目的开源组件目录。
```

## About 简介

```text
整理前端、后端、认证、数据库、AI、支付、部署、监控等开源组件，帮助新项目按模块快速组合技术栈。
```

## Website

第一阶段可以留空。后续如果增加 GitHub Pages 或搜索页面，再填写页面地址。

## 发布后检查

- README 能在仓库首页正确显示。
- Actions 中“校验组件目录”工作流通过。
- `LICENSE` 被 GitHub 正确识别为 MIT。
- `.github/CODEOWNERS` 已替换成真实负责人。
- Issues 页面能看到中文组件请求模板。
- Pull Request 页面能看到中文检查清单。
